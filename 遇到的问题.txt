2、在关联远程仓库的时候，输入$ git remote add origin git@github.com:2015800505zxf/'仓库名'.git  --> 提示 remote origin already exist 
解决办法： 
1、先删除远程 Git 仓库
$ git remote rm origin
2、再添加远程 Git 仓库
$ git remote add origin git@github.com:2015800505zxf/'仓库名'.git
